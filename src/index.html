Astroscrum
How toAPI Docs

Simple Slack powered scrum for your team

The Astroscrum bot collects scrum entries from your teammates over Slack, then distributes the summary to the entire team at noon.

Installing

Add dependency to package.json:

$ npm install --save hubot-astroscrum

Include package in Hubot's external-scripts.json:

["hubot-astroscrum"]

Configuration

Use crontab like syntax to set the scrum event times

# Default time to tell users to do their scrum
PROMPT_AT = "0 6 * * * *" # 6am everyday

# Default scrum reminder time
REMIND_AT = "30 11 * * * *" # 11:30am everyday

# Send the scrum at 10 am everyday
SUMMARY_AT = "0 12 * * * *" # noon

# Set to local timezone
TIMEZONE = "America/Los_Angeles"

Commands

We designed Astroscrum so that you have a conversation directly with the bot, not in a public room. We did this as an effort to cut out the noise of everyone chatting with the bot. The following examples assume you are sending direct messages to the bot (not Slackbot btw):

scrum help # learn about how to use astroscrum
scrum join # Join the scrum, this means you'll get dm'd by the bot daily
today I'll clean my desk # Tell the bot what you are doing today
yesterday I fixed email notifications
blocked waiting for @mmcdaris's approval on new designs
scrum summary # show a summary for the current scrum
